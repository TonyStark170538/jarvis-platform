# How to Add My Next Case Study

## Where the next case study will go

My next case study will be added to my existing portfolio in the same format as my J.A.R.V.I.S. project case study.

I will use the same three-part structure:

### 1. Problem

Explain the challenge, goal, or real-world problem the project was created to solve.

### 2. What I did

Describe my approach, tools, technologies, design decisions, and how I used AI during development.

### 3. What came of it

Show the final result, improvements achieved, lessons learned, screenshots, technical details, and the skills demonstrated.

The new case study will be added as a new project section alongside my previous work, keeping the same style, structure, and portfolio identity.

---

# Steps to Add a New Case Study

1. Create a new project folder in my portfolio.
2. Collect project information:

   * Project name
   * Problem being solved
   * Technologies used
   * Development process
   * Screenshots/demo links
   * Challenges and solutions
3. Write the case study using the three-beat structure:

   * Problem
   * What I did
   * What came of it
4. Review the text and make sure it matches my portfolio style and technical focus.
5. Add the project page to the portfolio navigation/projects section.
6. Update my project list and share the finished case study.
